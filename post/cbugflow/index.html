编译问题

1. fatal error C1083: 无法打开包括文件: “basetsd.h”: No such file or directory

  • 错误：

    fatal error C1083: 无法打开包括文件: “basetsd.h”: No such file or directory

  • 原因：缺少basetsd.h文件

  • 解决办法：安装windows 10 SDK文件即可解决

2. LINK : fatal error LNK1158: cannot run ‘rc.exe’

  • 错误

    LINK : fatal error LNK1158: cannot run 'rc.exe'

  • 原因：在PATH所提供的环境中搜索不到rc.exe程序

  • 解决办法：安装winsows 8.1 SDK，并将路径"C:\Program Files (x86)\Wexit()indows Kits\8.1\bin\x86"路径 添加到环境变量中。
