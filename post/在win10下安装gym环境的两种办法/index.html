在win10下安装gym环境的两种办法

November 20, 2019

Gym是强化学习中一个很重要的python库，里面包含了很多强化学习的环境。但比较可惜的是官方并没有提供Windows平台的支持，这对于习惯了Windows平台工作的强化学习研究人员来说很不方便，所以我找到了两个解决办法。

  • 直接在win 10上安装Gym环境
  • 利用WSL( Windows Subsystem for Linux )安装Gym环境

直接在win 10上安装Gym环境

安装 Microsoft Visual C++ 2017 Build Tools for Visual Studio 2017

  1. 下载链接 : Visual Studio
  2. 安装 Microsoft Visual C++ 2017 Build Tools

在Anaconda中创建python环境并安装相应的packages

  1. 在Anaconda中创建python环境

    conda create -n <your_env_name> python=3.7

  2. 如果需要在gym中录制视频，则需要安装ffmpeg

    conda install -c conda-forge ffmpeg

  3. 首先需要安装一个空的gym环境，因为在后面安装完全的gym环境的过程中会出现mujoco环境的错误，虽然这不影响其他模块的使用，但会导致gym不会被添加到环境中，也就没有办法import。

    pip install gym

  4. 安装pystan是ToyText环境的需要

    conda install pystan

  5. 这一步是安装gym环境成功的关键，国外大神kojoley成功地封装了atari模块使得安装这一补丁后，atari环境能够在win 10环境下成功运行

    conda install git
pip install git+https://github.com/Kojoley/atari-py.git

  6. 运行这一步是为了安装Box2D环境

    conda install swig
pip install Box2D

  7. 安装完整的gym环境，会有mujoco的错误出现，不过无所谓，不用在意

    pip install gym[all]

环境可视化(env.render())

  1. 下载Xming并按照提示安装

  2. 每次运行gym环境前，在win 10的cmd或者powershell中运行如下命令

    set DISPLAY=:0

Test

  1. 可以运行如下代码来测试

    import gym
   
env = gym.make('Pong-v0')
env.reset()
   
for i in range(1000):
   env.render()
   env.step(env.action_space.sample())

利用WSL( Windows Subsystem for Linux )安装Gym环境

安装 ubuntu WSL

  1. 控制面板 -> 程序 -> 启用或关闭windows功能 -> 适用于linux的windows子系统 或者在powershell中运行如下命令

    Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

  2. 去微软应用商店安装ubuntu 18.04的子系统，需要几分钟进行安装

在WSL中安装需要的软件包

  1. 运行如下命令先更新ubuntu的软件库

    sudo apt-get update && sudo apt-get upgrade -y && sudo apt-get upgrade -y && sudo apt-get dist-upgrade -y && sudo apt-get autoremove -y

可视化

  1. 跟前一种方法一样，在windows上下载Xming用来可视化WSL的图形画面

  2. 在ubuntu上进行画面输出的配置

    echo "export DISPLAY=localhost:0.0" >> ~/.bashrc
source ~/.bashrc
sudo apt-get install x11-apps
# start a demo application for test
xeyes

install anaconda and gym

  1. 下载Anaconda的安装脚本并进行安装

    cd /tmp
curl -O <url of anaconda>
chmod +x <the script of anaconda>
./<the script of anaconda>
source ~/.bashrc

  2. 创建python环境

    # Create the environment
conda create --name <env name> python=3.7
# Activate the environment
source activate <env name>

安装gym

   # 首先安装opengl---gym的需要
   sudo apt-get install python-opengl
   # 安装pip以便能够安装gym
   sudo apt-get install python3-pip # or python-pip
   # 安装gym
   pip3 install gym
   pip3 install gym[atari]
  1. 然后就可以写一个demo程序进行测试了

可听化

因为Xming不支持声音，所以WSL上有关声音的操作无法完成，比如flyppy-bird调用了pygame的声音模块导致其无法正常在wsl上运行，所以需要找到解决方案

  1. 在win 10 上下载 PulseAudio并解压

  2. 修改文件中的对应内容

| 文件路径及行数 | 变化 | 内容 | | ——————— | —- | ———————————————————— | | etc\pulse\default.pa | FROM | load-module module-waveout sink_name=output source_name=input | | 42 | TO | load-module module-waveout sink_name=output source_name=input record=0 | | etc\pulse\default.pa | FROM | #load-module module-native-protocol-tcp | | 61 | TO | load-module module-native-protocol-tcp auth-ip-acl=127.0.0.1 auth-anonymous=1 | | etc\pulse\daemon.conf | FROM | ; exit-idle-time = 20 | | 39 | TO | exit-idle-time = -1 |

  1. 在WSL上设置 pulseaudio

    sudo add-apt-repository ppa:therealkenc/wsl-pulseaudio
sudo apt-get update
   
sudo apt-get install pulseaudio

    安装完成无误后，运行如下配置命令使得PulseAudio能够通过TCP运行

    echo "export PULSE_SERVER=tcp:localhost" >> ~/.bashrc
source ~/.bashrc

  2. 在cmd或者powershell中运行bin文件夹下的 pulseaudio.exe

gym环境完整性测试程序

import time
import gym

test_envs={'algorithm':'Copy-v0',
           'toy_text':'FrozenLake-v0',
           'control':'CartPole-v0',  
           'atari':'Pong-v0',
           'mujoco':'Humanoid-v1',    
           'box2d':'LunarLander-v2' }

for k,v in test_envs.items():

    try:
    	env = gym.make(v)
    except Error:
        print("failed to run {} environment".format(k))
        continue
        
    num_episodes = 1

    for i in range(num_episodes):
        state = env.reset()
        totalReward = 0

        for _ in range(100):
            env.render()

            # take a random action
            randomAction = env.action_space.sample()
            observation,reward,done,info = env.step(randomAction) 

            time.sleep(0.1)
            totalReward += reward

        print("success to run {} environment".format(k))

    env.close()
